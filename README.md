# Connect Four Game (C++)

A console-based Connect Four game developed in C++ as a Programming Fundamentals project.

## Features

- Two-player gameplay
- Random toss system
- Horizontal, vertical, and diagonal win detection
- Match record storage using file handling
- Interactive menu system

## Technologies Used

- C++
- File Handling
- Object-Oriented Programming Concepts
